# ai-berkshire Roadmap

## P0：近期（1-2个月）

### A股数据源接入（部分完成）
- ✅ 基础行情/财务已接入：`tools/ashare_data.py`（腾讯行情 + 东方财富，零依赖直连）
- 待完成：龙虎榜数据、筛选器与各 Skill 工作流的全面整合

## P1：中期（3-6个月）

### HTML 报告输出
- 在 Markdown 基础上增加 HTML 报告格式
- 支持暗色模式、导航栏、图表可视化
- 提升报告传播性和阅读体验

### 多档深度模式
- `lite`：5分钟速判，快速给出估值区间和核心结论
- `standard`：当前默认模式，完整多Agent研究
- `deep`：增加更多交叉验证和历史类比，机构级深度

### 多股横向对比
- 支持 2-4 只股票同维度横向对决
- 同行业公司估值对标
- 输出对比矩阵和择优建议

## P2：长期（6个月+）

- 为 Skill 输出增加回归测试
- 确保迭代不破坏已有功能

### 组合级分析
- 持仓组合健康度评估
- 行业/地域集中度分析
- 相关性风险检测
