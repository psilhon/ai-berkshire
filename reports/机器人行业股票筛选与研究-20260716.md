# 机器人行业股票筛选与研究

> 数据截止：2026-07-16。范围假设：沪深A股机器人产业链。本文是学习与研究材料，不构成投资建议。

## 结论先行

机器人不是一个单一行业，而是四类生意的集合：工业自动化、核心零部件、消费机器人、移动/服务机器人。真正值得研究的不是“最像机器人”的股票，而是已经有真实客户、现金流和利润，再把机器人作为增量的公司。

当前优先级：

1. **汇川技术**：工业自动化基本盘最强，机器人是增量期权；适合做核心研究对象，但当前估值不便宜。
2. **科沃斯**：真正有规模的家庭机器人产品，业绩修复后估值相对可算；需观察利润率和竞争。
3. **九号公司**：产品、渠道和增长较强，包含割草机器人等新曲线；但本质仍是“短交通+机器人产品平台”，不是纯机器人股。
4. **双环传动、柯力传感**：零部件/传感器方向质量较稳，机器人弹性低于纯题材股，但估值和现金流更容易研究。
5. **三花智控**：公司质量高，但机器人业务目前更像长期期权；以当前估值买入，必须相信机器人零部件未来量产。

**核心判断：机器人赛道值得研究，但当前A股多数“人形机器人概念股”并未通过价值投资的第一轮排除。优先选择已有盈利和现金流的自动化、消费机器人与零部件龙头，不追逐尚未验证订单和利润的纯概念。**

## 第一阶段：investment-checklist 六关初筛

### 行业能力圈判断

机器人行业可以理解，但不能用一句话概括所有公司。工业机器人靠控制器、伺服、电机、减速器和系统集成赚钱；消费机器人靠硬件销售、品牌、软件和渠道赚钱；人形机器人目前仍处于客户验证和量产前阶段。

因此，本次把“机器人股票”拆成：

- 工业自动化：汇川技术、埃斯顿、机器人、拓斯达、先导智能；
- 执行器/传动/传感器：绿的谐波、双环传动、中大力德、鸣志电器、柯力传感、兆威机电；
- 具身感知与系统：江苏北人、奥比中光；
- 机器人相关平台与消费机器人：三花智控、拓普集团、石头科技、科沃斯、九号公司。

## 第三阶段：investment-research 深度比较

### 1. 汇川技术（300124）：最适合做核心研究，但不是纯机器人股

**生意本质**：为制造业提供变频器、伺服、PLC、控制系统和新能源汽车电驱等自动化基础设施，机器人业务是控制、电机、执行器能力的延伸。

2025年营收451.05亿元、归母净利润50.50亿元、ROE16.34%、毛利率28.95%；2021—2025年收入从179.43亿元增长至451.05亿元，净利润从35.73亿元增至50.50亿元。2025年经营现金流约66.81亿元，高于归母净利润。公司2025年度拟每10股派5元。[2025年报披露](https://vip.stock.finance.sina.com.cn/corp/view/vCB_AllBulletinDetail.php?id=12222830&stockid=300124)

当前价格60.91元、动态PE约34.79倍、PB约4.71倍；以2025年EPS和每股净资产精确计算，PE约32.57倍、PB约4.66倍。三年情景模型为：乐观85.3元、中性62.2元、悲观33.7元。中性情景几乎没有安全边际。

护城河来自客户导入、控制算法、产品矩阵、研发规模和本土服务网络；机器人故事不是全部估值基础，真正的底盘是工业自动化和新能源车电驱。风险是制造业资本开支、价格竞争、应收账款和机器人业务投入回报不及预期。

**判断：优质公司，合理偏贵。** 研究价位可放在55元以下；60—70元更适合等待验证，不适合把机器人想象空间一次性买满。

### 2. 三花智控（002050）：高质量零部件平台，机器人期权估值较高

**生意本质**：通过制冷空调控制元件和汽车热管理零部件，为家电、新能源汽车及未来机器人提供关键机电部件。

2025年营收310.12亿元、归母净利润40.63亿元、ROE15.80%；制冷空调零部件收入185.85亿元，汽车零部件124.27亿元。2026Q1营收77.74亿元，同比+1.36%，归母净利润9.28亿元，同比+2.68%，扣非净利润同比增长15.52%。[2025年报](https://disc.static.szse.cn/disc/disk03/finalpage/2026-03-24/80d26a66-2c66-4eaa-bca0-f8a1a8f261a0.PDF)、[2026Q1数据](https://money.finance.sina.com.cn/corp/view/vCB_AllBulletinDetail.php?id=12285147&stockid=002050)

当前价格42.10元；按2025 EPS和BPS计算，PE约40.87倍、PB约5.58倍。三年情景模型的中性目标约38.4元，低于当前价格；乐观目标62.3元依赖20%利润增速和35倍PE。

其机器人逻辑更接近“核心零部件平台向人形机器人延伸”，但现阶段机器人收入对合并报表的解释力有限。真正的投资问题不是“能不能造机器人零件”，而是机器人客户能否从样件验证进入稳定量产。

**判断：公司好，价格不便宜。** 45元以上需要机器人量产证据；35元附近才有更好的安全边际。

### 3. 科沃斯（603486）：真实机器人收入最清晰，但竞争决定利润率

**生意本质**：用导航、感知、清洁算法、品牌和渠道，把扫地、洗地、擦窗、割草等家务自动化产品卖给家庭用户。

2025年营收190.40亿元，同比+15.10%；归母净利润17.58亿元，同比+118.13%；ROE22.19%、毛利率48.8%。2026Q1营收49.02亿元，同比+27.06%，归母净利润4.05亿元，同比下降14.73%，扣非归母净利润同比增长11.46%。这说明收入增长尚可，但表观利润仍受费用、汇率和竞争影响。[2025年报与2026Q1数据](https://stock.finance.sina.com.cn/stock/go.php/vReport_Show/kind/company/rptid/830567580312/index.phtml)

当前价格55.72元；按2025 EPS/BPS计算，PE约17.97倍、PB约3.61倍，明显比三花和绿的谐波更可算。三年情景模型为：悲观43.4元、中性74.3元、乐观128.6元。模型区间很宽，核心变量是利润率和海外增长，而不是销量单一增长。

护城河是品牌、产品体验、算法和渠道，但转换成本不高，石头、追觅、米家等会持续压价。科沃斯的优势是双品牌和多品类，弱点是历史利润波动较大。

**判断：机器人产品属性最纯、估值相对合理。** 55元附近可以进入研究/分批观察区；若Q1后利润率继续下滑，则低PE可能是盈利高点陷阱。

### 4. 九号公司（689009）：平台化产品公司，机器人新曲线有价值但不能忽略主业

**生意本质**：用智能硬件、软件、供应链和品牌，把两轮车、滑板车、平衡车、全地形车和割草机器人卖给全球消费者。

2025年营收212.78亿元，同比+49.89%；归母净利润17.58亿元，同比+62.17%；ROE26.08%。2026Q1营收58.70亿元，同比+14.82%，归母净利润2.03亿元，同比下降55.42%，主要受汇兑损失影响；报告期割草机器人和电动两轮车是重要观察方向。[2026Q1数据](https://emweb.eastmoney.com/PC_HSF10/OperationsRequired/Index?code=SH689009&type=web)

当前价格41.30元；按2025 EPS/BPS计算，PE约16.71倍、PB约4.15倍。三年情景模型为：悲观40.1元、中性67.7元、乐观120.7元。当前估值对增长仍有要求，但比纯人形机器人概念更有财务锚。

护城河是产品定义、品牌、渠道、软件和供应链协同；风险是海外汇率、贸易政策、两轮车价格竞争和新品类投入。它不是纯机器人股，购买理由不能只写“割草机器人增长”。

**判断：成长质量较好，但波动和业务复杂度高。** 35—42元区间值得研究；若重新站上55元，需要利润增长重新加速才能支撑。

## 行业风险与逆向判断

| 风险路径 | 影响 | 观察指标 |
|---|---|---|
| 人形机器人量产延期 | 高估值零部件股大幅回撤 | 客户定点、批量订单、单机BOM、毛利率 |
| 专用自动化替代人形机器人 | 中高 | 工厂真实部署的投资回收期与复购 |
| 工业自动化价格战 | 中高 | 汇川、埃斯顿、海外厂商的毛利率和份额 |
| 消费机器人品牌价格战 | 高 | 科沃斯/石头ASP、毛利率、销售费用率 |
| 海外贸易与汇率 | 中高 | 九号、石头、科沃斯海外收入和汇兑损益 |
| 题材资金退潮 | 高 | 纯概念股成交、融资余额、估值与业绩脱钩程度 |

最强空方论点：机器人行业的技术趋势大概率成立，但产业价值最终可能被下游客户、AI平台和规模化制造拿走；先发公司未必是最终赢家，零部件也可能因为国产替代和价格战而从高毛利变成普通制造业。

最可能的分析错误：用“行业TAM很大”替代“公司现在能赚多少钱”。

## 最终决策

| 层级 | 股票 | 结论 |
|---|---|---|
| 核心研究 | 汇川技术 | 最强工业自动化底盘，机器人增量，等待估值回落或盈利上修 |
| 核心研究 | 科沃斯 | 真实机器人产品和估值较匹配，重点跟踪利润率与海外竞争 |
| 成长观察 | 九号公司 | 增长与产品力强，但业务复杂、汇率和海外风险较高 |

| 价值观察 | 双环传动、柯力传感 | 财务质量较稳，机器人弹性有限，需单独研究估值与订单 |
| 暂不参与 | 绿的谐波、埃斯顿、机器人等 | 技术/题材可跟踪，但当前财务或估值未通过第一轮排除 |

### 买入前必须补齐的证据

1. 汇川机器人相关收入、订单和毛利率是否已经单独达到可见规模。
2. 三花机器人执行器是否从样品/验证进入量产，客户和单机价值量是多少。
3. 科沃斯Q1利润下降是季节性/费用投入，还是竞争导致的结构性压价。

## AI分析置信度与投资确定性

2025年财务数据、2026Q1部分数据和当前估值快照置信度较高；行业TAM、人形机器人量产节奏和未来估值倍数置信度中低。汇川、科沃斯、九号的“已有业务可研究”确定性高于纯人形机器人概念股，但不代表短期股价上涨确定。
